CAJAS

Una caja es cualquier objeto en HTML que envuelva a otro y le permita
tener margenes y flotar.

Para que una caja esté dentro del texto, se usa inline 

#objeto
{
display:inline; Se mezcla con el texto, con margenes superiores e inf.
display:block;  
display:inline-block;
}

div, figure, header, section, article, p, nav --> BLOCK
audio, video, img span --> INLINE
